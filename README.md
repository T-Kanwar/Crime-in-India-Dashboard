# Crime-in-India-Dashboard
🚨 Crime Dataset India Dashboard (Power BI)

An interactive Power BI dashboard designed to analyze crime patterns across major cities in India.
This project explores crime trends, victim demographics, weapon usage, and case closure rates using data visualization and DAX analytics.

The dashboard helps users quickly identify crime hotspots, dominant crime categories, and yearly crime trends through dynamic charts, maps, and KPI indicators.

🔍 Insights from the Dashboard

Other Crime accounts for the largest share of reported crimes.

Burglary is the most frequently occurring crime type.

Major metropolitan cities like Delhi and Mumbai show higher crime counts.

A large portion of crimes involve weapons such as knives or explosives.

Crime patterns fluctuate slightly across months but remain relatively consistent annually.
